# Constructora_App
Contructora APPDebido al crecimiento de la constructora J&amp;J, se tiene la necesidad de desarrollar un software que permita, administrar todo el proceso de registro y venta de un departamento en un proyecto inmobiliario. El cliente desde la página web donde se muestra fotos, videos, información, etc sobre el proyecto enviará una solicitud para brindar mayor información. Esta solicitud será recibida por un asesor de ventas quien se contará con el cliente para brindarle toda la información y guiar en el proceso de compra.
